一、前期调研分析

在大一上学期我们学习了c语言，但由于缺少系统的学习网站，因此在学习期间遇到了许多困难。因此在本次小学期中，我们小组计划制作一个面向编程初学者的C/C++编程学习网站，涵盖学习资料、开发工具等内容，提供一站式服务。

二、建站的目的及功能定位

该项目是制作一个面向编程初学者的C/C++编程学习网站，以帮助初学者学习掌握C/C++与编程知识为目的。

三、网站技术解决方案

该项目网站主要面向PC端用户，可在Chrome、edge等主流PC浏览器上运行。网站将在VScode上基于html5进行编写，实现链接等功能，再以CSS对网站背景风格，字体等进行美工，基于JavaScript来实现网页与用户的交互如实现注册功能等。

四、网站内容规划

hat’s up C/C++学习网站将包含对C/C++发展历史的介绍、C/C++的编程教程参考、C/C++函数库的参考、C/C++编程软件的介绍以及对其他相关学习资源网站与在线课堂的链接，同时还提供分为基础和进阶两种难度的在线题库练习。

网站将具备用户注册、登录并保存或清除用户信息的功能、提供目录并跳转到相应网页的功能、提供链接并跳转到外部网站的功能、在浏览器中展示图片与播放视频的功能、提供在线练习的功能等。

网站将包含六个主要页面，主要页面下包含共计二十多个分页面。

主要页面分别有编程语言简介板块、开发平台板块、学习资源板块、C/C++参考板块、训练题单板块和网站制作组介绍板块。

其中，语言简介页面下设C/C++的简介和发展历史两个分页面；开发平台页面下设Dev C++、Visula Studio、Visual Studio Code、Code::Blocks和Xcode这五个分页面；学习资源页面包含编程教程、热门博客和学习数据模块；；C/C++参考页面将包含C函数库和C++函数库两个分页面，分页面下各分六个子页面对各头文件进行介绍；训练题单页面将分别设置基础题库和进阶题库两个分页面；关于我们页面将包含项目介绍和成员简介两个分页面。
