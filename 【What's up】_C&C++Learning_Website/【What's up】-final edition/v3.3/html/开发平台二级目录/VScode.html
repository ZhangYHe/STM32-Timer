下载地址：点击下载

1.首先，到vs cdoe官网下载vs code，选择windows版本

2.双击打开安装包

3.同意协议，然后下一步。

4.选择安装路径，继续下一步

5.创建菜单文件夹(相当于在开始里面添加了快捷方式，直接下一步)

6.创建桌面快捷方式，并添加PYTH，之后下一步安装，

7.打开vs code后，侧边栏第五个图标，打开插件，搜索c，点击install安装C/C++插件。

8.进入ming-w64

9.点击download进入下载页面

10.点击SourceForge，进入托管网站

11.滚动到页面最下面可以看到exe版本的安装包，点击下载

12.下载完成后双击打开安装包，(可能会弹出用户账户控制提示，选择是即可)，点击下一步

13.Architecture选择x86_64(因为现在基本都64位操作系统)

14.选择安装路径，继续点击下一步，会开始下载相应的安装文件

几点注意：

1.MinWG，vscode安装路径最好全为英文，否则可能报错。

2.vscode文件夹，以及.c文件所在的路径最好也全为英文，否则依旧可能报错。

3.新建的三个json文件，注意里面的路径一定要配置为自己所安装软件的对应路径，否则运行时肯定报错。

4.运行程序时，比如运行hello.c文件，那么当前编辑窗口必须是hello.c。

5.如果gcc -v命令失效，用管理员身份打开cmd窗口，再运行gcc -v命令。

6.生成的exe文件，如果直接鼠标双击运行，可能闪一下就消失了，那是因为没有加入暂停命令。

7.vscode，minwg安装完后注意先重启一下电脑，然后再进行后面的步骤。

8.如果单纯只是编译简单的c文件，可以不用安装vscode，vscode只是可视化更好，相当于给MinWG套了个壳，让它更好看，直接用命令行也可以编译。
