下载地址：点击下载

1.必应搜索 dev c++，选择一个网站下载安装包（如下图被标记的候选项）：

2.点击绿色的图标download后，等待5秒钟，然后会弹出下载窗口

3.选择保存

4.下载完成后选择运行

5.如果弹出这个窗口，选择是

6.选择English，然后单击ok

7.单击I Agree

8.单击Next

9.选择好安装路径后，单击Install安装（我这里选择的安装路径是F盘）

10.单击Finish结束安装

11.在弹出的窗口里选择简体中文，然后单击Next

12.继续单击Next

13.单击ok，结束安装和配置

14.新建源代码

15.编译运行程序

16.选择一个路径保存要编译的代码

17.运行结果
