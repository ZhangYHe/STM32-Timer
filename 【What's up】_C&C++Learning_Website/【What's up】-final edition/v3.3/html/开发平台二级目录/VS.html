下载地址：点击下载

1.下载完成后，我们双击下载的软件开始安装

2.在弹出的画面点击继续

3.等待完成然后我们开始配置开发环境

4.如果不想用默认的安装路径（默认是安装在C盘的），可以点击更改来指定安装路径

5.选择好安装路径后，在右下角安装按钮的左边选择下载时安装（默认就是，这样可以节省安装时间），然后点击安装

6.等待安装完后，点击以后再说，也可以注册一个账号登录

7.开发设置选择Visual C++

8.根据个人喜欢选择主题，然后点击启动。

9.点击创建新项目

10.选在空项目，然后点击下一步

11.填写项目名称，和项目存放的位置，然后点击右下角的创建

12.在源文件上右击添加-->新建项，也可以直接使用快捷键ctrl+shift+A

13.填写程序名称并将后缀名改为.c，然后点击添加

14.接下来运行刚刚写的代码，验证一下环境是否OK，点击调试-->开始执行（不调试）或者直接快捷键CTRL+F5
