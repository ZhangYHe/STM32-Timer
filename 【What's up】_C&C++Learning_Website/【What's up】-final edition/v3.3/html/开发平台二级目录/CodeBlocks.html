下载地址：点击下载

1.在浏览器上搜索CodeBlocks官网或者直接输入网址http://www.codeblocks.org/ 进入CodeBlocks官网。

2.进入下载页面Downloads,一般都会选择第一个Download the binary release(二进制版本）

3.进入之后，就会出来很多版本，我们要选择一个合适的版本，一般都会下载自带编译器的版本

4.点击之后可以直接点击direct link下载，或者点击mirror手动选择下载地址。

5.下载完成后启动安装程序

6.注意：这里会有一个问题，如果这里你像我一样改变了默认安装地址，将安装地址选到了E盘，之后 安装完成后需要改一下编译器位置。

7.根据需要看看是否要立即运行。

8.安装完成。

9.打开CodeBlocks，如果有问题的的话，在右下角会出现一个问题，如图：

10.选择程序目录setting->Compiler 然后出现下图，按照图示点击，选择自己安装的位置，找到MinGW文件夹，点击确定。

11.最后点击OK，这样就完成了。程序就可以编译运行了。
