下载地址：点击下载

1.点击"Install App"，安装Xcode

2.安装好之后，打开XCode，点击"File" -> "New" -> "Project..."

3.选择"Single View Application"，点击"Next"

4.填写项目相关信息，点击"Next"

5.选择存放项目的位置，点击"Create"

6.右下角的对象库中选择Label控件，将其拖曳到View设计界面上，双击Label控件，在其中输入"Hello World"

7.右击选择Add Files to "HelloWorld"

8.将一张图片添加到工程

9.选择Image View，点击右边属性检查器，Image下拉列表中选择上一步添加的图片

10.运行模拟器，可以切换不同设备类型
